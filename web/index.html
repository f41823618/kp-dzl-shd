Invoice Summary Studio

对账单一键汇总成开票清单

直接把多个 Excel 拖进来，浏览器本地完成识别、汇总、预览和导出。 不依赖后端，适合挂到 GitHub Pages 持续部署。

当前规则
  • 自动识别不同工作表的表头位置和列顺序
  • 支持多文件、多工作表合并汇总
  • 井座、闸阀方井按“数量 × 高度”换算米数
  • 空行、页脚合计行自动过滤

支持 .xlsx / .xls，多选后统一汇总。
